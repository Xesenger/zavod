# ZAVOD Operations Notes

Operational notes live here when they are useful for current work but are not
canon, role truth, or a production plan.

## Folders

- `projects-workbench/` - active notes for the Projects operational surface and
  shared web-engine discipline.

## Boundary

Operations notes may guide implementation work, but they do not override:

- `docs/canon/`
- `docs/roles/`
- `docs/plans/`
