# Legacy Docs

This folder contains historical material retained for archaeology.

Legacy docs may include:

- old migration plans
- retired reference templates
- checkpoint handoffs
- closed audit notes
- prototype HTML or UI concepts

Legacy material is not current truth. If a legacy document conflicts with
`docs/canon/`, `docs/roles/`, `README.md`, or `ROADMAP.md`, the current active
documents win.

## Folders

| Folder | Contents |
|---|---|
| `checkpoints/` | Historical implementation checkpoints |
| `projects-web-migration/` | Archived Projects web migration notes and prototypes |
| `plans/` | Frozen or superseded plans preserved as references |
| `reference/` | Retired project-template reference material |
| `audits/` | Closed audit dumps kept out of active docs |
