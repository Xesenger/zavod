# Shifts

## Role
Shift execution record layer.

## Contains
- execution records
- lifecycle records
- handoff records

## Rule
Shift executes task.
Shift does not redefine task.
Shift output must be traceable to a single task.
Shift is execution record only.
Shift does not define project truth.
Only `/project` contains active truth.
All other layers are non-authoritative.
