# Legacy Audits

Closed external audit dumps can be kept here locally when they are useful for
short-term review.

This folder intentionally ignores raw `.txt`, `.log`, and `.tmp` dumps so large
or one-off audit transcripts do not become repository truth by accident.

If an audit produces durable project guidance, summarize it into:

- `ROADMAP.md`
- `README.md`
- `docs/plans/`
- `docs/canon/`

Do not treat raw audit dumps as active documentation.
